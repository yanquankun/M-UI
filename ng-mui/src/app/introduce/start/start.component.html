此项目为我个人设计UI框架，不可用于商业用途，版本为最初始版本，存在较多问题，会在后续版本中不断更新迭代，此版本并未投入实际生产环境中使用。
开发此UI框架，意味自己独立造轮子，且后期可以投入项目正常使用，对个人、公司、团队中的UI框架增加较强的定制性
github地址：项目初期，暂不开源，项目完善后，将会进行开源
npm地址：https://www.npmjs.com/package/m-ui-lib
注意：需引入全局样式
全局less引入：在 angular.json 文件中引入mui样式: { "styles": [ "node_modules/m-ui-lib/mui.less" ] }
"或全局css引入：在 angular.json 文件中引入mui样式: { "styles": [ "node_modules/m-ui-lib/mui.css" ] }
公用样式文件（如：tag控件烟花特效样式）： output.css output.less output.min.css output.min.less
